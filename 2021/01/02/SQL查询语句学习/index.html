高级查询

多表联查

  1. 笛卡尔积查询
    select 列名列表 from 表1,表2;
  2. 内连接查询
    select 列名列表 from 表1,表2 where 两表关联列条件表达式;
    select 列名列表 from 表1 inner join on 两表关联列条件表达式;
  3. 外链接查询
    1. 左外连接查询
      以左边的表为标准来连接查询右边的表，如果右边的表的数据能满足，就正常显示；如果右边的表不能满足就填充为null
    2. 右外连接查询
      和左外连接查询的逻辑相同，只是以右边的表为查询标准
  4. 全连接查询

子查询

    子查询

  3. select 子句子查询：将一个查询的结果和其他查询的结果进行组合

小结

能用多表联查方式实现的查询使用子查询一定能够实现，但使用子查询实现的查询使用多表联查不一定能够实现。
